Hu·man | Humanist Documentation

I have a natural revulsion to any operating system that shows so little planning as to have to named all of its commands after digestive noises (awk, grep, fsck, nroff).
— Unknown

UNIX was built by hackers. For hackers, the source is the documentation. man was designed to avoid straight answers. So how do I list hidden files in a directory with ls? What's grep? How do I get out of less? Developers waste precious development time Googling around for questions that should have obvious answers.

Enter Human. Human believes in instant code snippets. No more following dozens of failed StackOverflow links, or scrolling through pages of man. Just replace it with human command and get an FAQ written by and for developers like you.

Install

Using Sudo (root priviledges needed): 
sudo curl "http://humanual-sv.appspot.com/static/install" -o /usr/bin/human
As a non-root user: 
cd ~/
echo "PATH=\"$PATH:~/human\"" >> ~/.profile
mkdir human
cd human
curl "http://humanual-sv.appspot.com/static/install" -o human
chmod +x ~/human/human
